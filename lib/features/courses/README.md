# ميزة الدورات (Courses)

هيكل **Clean Architecture** لشاشة المعرض والدورات في منصة Algonaid.

## الطبقات

| المجلد | المحتوى |
|--------|---------|
| `domain/` | كيانات، عقود مستودعات، حالات استخدام — بدون Flutter أو I/O |
| `data/` | نماذج JSON، مصادر بيانات، تطبيق المستودع |
| `presentation/` | صفحات، ويدجت، مزودي حالة (عند الحاجة) |

## التبعية

- `domain` لا يعتمد على طبقات أخرى في الميزة.
- `data` يعتمد على `domain` و`core` (شبكة، ثوابت).
- `presentation` يعتمد على `domain` (و`data` فقط عبر الحقن / المستودع).
